> [!Important]
> 此版本插件代码进行了大范围重构，如遇安装/更新错误，请先卸载插件后重新安装。

## 本体修改

- 新增 `设备识别凭证`, 相较于"机器识别凭证", 其基于更稳定、现代化的设备唯一标识符识别生成算法, 在多数场景下变更概率更低。但目前该功能仍处于早期测试阶段，欢迎积极反馈，帮助我们进一步完善相关算法
- 修正了过时的模块反馈链接
- 优化了模块加载与编译逻辑
- 调整了打断热键的逻辑, 需要重新配置
- 修复了打断热键显示文本仅显示键盘热键未显示手柄按键的问题
- 调整了本地化功能的内部逻辑, 需要重新配置
- 删除了 `更新翻译` 功能与操作按钮, 避免提前引入不必要的本地化文件变更
- 优化了模块配置界面卡片组件的视觉效果
- 调整了命令参数错误时的提示信息外观
