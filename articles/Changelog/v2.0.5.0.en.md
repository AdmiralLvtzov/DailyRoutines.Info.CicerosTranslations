## Framework Modifications

- The calculation logic related to coordinates and hitboxes have been greatly improved.
- An issue of missing tray notifications has been fixed.
- The behaviours of connection, retry and backoff strategies have been adjusted in accordance with official recommendations from Universalis. 
- The rendering logic of module configuration interfaces has been adjusted. Now exceptions occurred during the rendering process would no longer jeopardize the rendering of the framework interface.

## Module Modifications

### Auto Retainer Work [UI_Operation]

- An issue that users were unable to add new item configurations for `Auto Retainers Price Adjust` has been fixed.
- The logic of interface rendering has been adjusted.

### Better Blacklist [System]

- An issue that, it's unable to acquire player names in Blacklist if users access the module configurations while not logged in, has been fixed.
- The drawing logic of the configuration interface has been fully refactored.
- An issue that Blacklist lists were not preloaded in advance like shared Blacklist groups has been fixed.

### Occult Crescent Helper [Assistance]

- The drawing logic of the Phantom Job interface has been adjusted.
- Now the character would dismount 1.5 to 2 seconds later after entering a FATE area.
- The configuration option `CE Commencement Notification` has been separated into three options `Send Notification`, `Send TTS Notification` and `Play System Sound Effect`. All the options are enabled by default.
- New configuration option `Auto Dismount After Entering FATE Area`. Enabled by default.

### Better Market Board [UI_Optimization]

- The combo box of `Data Center` and `Server` was relocated to the left sidebar.
- Now `Auto Display Available NPC Vendor` is a prerequisite module.
- A button to the interface of related NPC vendors was introduced while checking the availability of items at NPC vendors.
- If an item could be purchased both at NPC vendors using `gil` and at the market board, a red line would present in the price list to indicate deals priced higher than the price at NPC vendors.

![BetterMarketBoard-NPCPrice](/assets/Changelog/2.0.5.0/BetterMarketBoard-NPCPrice.png)

- The cap of market history requests has been increased to 1000 according to official recommendations from Universalis.
- New view `Market Trend` has been introduced for checking the price trends of specific items.

### Instant Character Acceleration [System]

- `Auto Speed Multiplier` is no longer a prerequisite module.

### Customize Entry Count In Party Finder [Recruitment]

- Adaption work for Patch 7.5 has been introduced.

### Better Classes And Jobs Interface [UI_Optimization]

- Adaption work for Patch 7.5 has been introduced.

### Auto Open Treasure Coffer [Combat]

- The implementation has been adjusted.

### Better Duty Finder Registration [Recruitment]

- Adaption work for Patch 7.5 has been introduced.
- The size and positioning logic of buttons have been optimized.
