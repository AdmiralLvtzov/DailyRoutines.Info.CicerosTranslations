## 本体修改

- 大幅优化了各类坐标、地面、碰撞判断计算逻辑
- 修复了托盘通知失效的问题
- 根据 Universalis 官方要求, 调整了连接、重试与退避策略
- 调整了模块配置界面渲染逻辑, 现在模块自身渲染异常不会再溢出至插件窗口

## 模块修改

### 自动雇员作业 [界面操作]

- 修复了无法添加自动雇员改价新物品配置的问题
- 调整了界面渲染逻辑

### 更好的黑名单 [系统]

- 修复了在未登录状态下打开模块配置界面时, 会因本机数据不足导致无法获取黑名单内玩家名称的问题
- 完全重构了配置界面绘制逻辑
- 修复了黑名单列表未像共享黑名单组一样提前进行数据预热的问题

### 蜃景幻界新月岛助手 [辅助]

- 调整了辅助职业列表界面的绘制逻辑
- 现在在自动进入危命任务范围时, 会等待 1.5 秒至 2 秒再下坐骑
- 配置项 `通知 CE 开始` 拆分为 `发送通知消息`、`发送语音消息` 与 `发送系统音效`, 均默认启用
- 新增配置项 `进入事件范围时, 自动下坐骑`, 默认启用

### 更好的市场布告板 [界面优化]

- 调整 `地区选择` 与 `服务器选择` 组合框至左侧边栏顶部
- 新增前置模块 `自动显示物品 NPC 商店信息`
- 在搜索物品存在 NPC 获取途径时, 会显示一个获取途径界面快捷打开按钮
- 在搜索物品存在 NPC 获取途径, 并可通过 `金币` 购买时, 会在价格列表中划出一道红色的分隔线, 突出市场中高于 NPC 售价的物品

![BetterMarketBoard-NPCPrice](/assets/Changelog/2.0.5.0/BetterMarketBoard-NPCPrice.png)

- 根据 Universalis API 推荐, 将物品历史交易数据条数请求扩大到了最多 1000 条
- 新增 `价格走势` 视图, 支持查看物品历史价格走势

### 即刻全速 [系统]

- 删除了前置模块 `自动调整移动速度` 要求
