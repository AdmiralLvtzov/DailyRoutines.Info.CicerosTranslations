## Framework Modifications

- An issue of being unable to access `The Firmament` in some automated processes has been fixed.

## New Modules

### NVIDIA DLSS Identifiable Client [System]

Make non-global clients (in other words, non-NA/EU/JP/OCE clients) identifiable for the DLSS optimization feature in NVIDIA App, allowing users to enable DLSS optimization for the client.

![AutoEnableDLSSSupport](/assets/Changelog/2.0.6.0/AutoEnableDLSSSupport.png)

## Removed Modules

### Higher Quick Synthesis Cap [System]

The module has been merged into the module `Better Crafting Log`.

## Module Modifications

### Instant Logout Or Exit [System]

- The module title has been adjusted. The previous title was `Instant Logout`.
- Now the module would work on all the following scenarios:
    - Clicking `Exit Game` in the title screen.
    - Ending the process via the title bar of the game window.
    - Clicking `Exit Game` in System Menu of the game.
    - Ending the process via the task bar.
- The module is now a `recommended module`.

### Auto Fishing Status Persist [System]

- The title, description and internal name of the module have been adjusted. The previous title was `Block Input During Fishing`. Users are required to re-enable the module after the update.
- The implementation has been fully refactored, adopting a more underlying logic.

### Auto Block Unnecessary Pop-ups [UI_Optimization]

- The blocking effect has been improved.
- More unnecessary pop-ups have been blocked.

### Auto Confirm Materia Melding And Retrieval [General]

- The melding efficiency has been improved.

### Auto Check Item Level [Combat]

- An issue that the module would work in Duty Recorder has been fixed.

### Better Reward Interface [UI_Optimization]

- The interface appearance and interaction animations have been fully refactored.

![OptimizedFreeShop-Addon](/assets/Changelog/2.0.6.0/OptimizedFreeShop-Addon.png)

### Better World Visit [System]

- Now all the servers within a data center are sorted lexicographically.

### Instant Character Acceleration [System]

- The logic has been optimized and the compatibility has been improved.

### Auto Confirm Quest Accepting [UI_Operation]

- An issue that the module didn't work has been fixed.
- The interrupt hotkey would no longer work on the module.

### Auto Skip Cutscenes [System]

- Adaption work for Patch 7.5 has been introduced.
- Now the `inspection` and `dart shooting` scenarios could be automated or skipped.
- More backup logic has been introduced to make sure the module work in all the applicable scenarios.

### Better Market Board [UI_Optimization]

- New UI component has been introduced, indicating the top three servers with the highest prices for the item and also the top three with the lowest prices.
    - To access the full detailed price information across all the servers and data centers, click `Show All` in the upper right corner.

![BetterMarketBoard-Overlay](/assets/Changelog/2.0.6.0/BetterMarketBoard-Overlay.png)

- The `HQ`-only filter could be enabled on the right.
- An issue that some unexpected behaviours regarding `HQ` items in Universalis mode has been fixed.
- Now the status of the `HQ`-only filter would persist regardless of the current item.

### Better Crafting Log [UI_Optimization]

- The following features no longer require the plugin `Raphael.Dalamud`:
    - Showing the same recipe under other jobs
    - Material source tracing
    - Job switching
- Only one of the `Switch Job` button and the vanilla `Synthesize` button would present, depending on the circumstances.
- The appearance of the feature `Same Recipe Under Other Jobs` has been improved. The corresponding job name would present while hovering an icon.
- The target job name would present while hovering over the `Switch Job` button.
- All the features could be enabled or disabled separately in the configurations.
- New feature `Higher Quick Synthesis Cap`, to increase the cap of quick synthesis from 99 to 255. Enabled by default.
- New feature `Higher Quality Cap In Trial`, to increase the quality cap in trial synthesis from a half of the corresponding recipe to full. Enabled by default.

### Occult Crescent Helper [Assistance]

- The appearance of `Phantom Job List` has been adjusted.
- The word `Phantom` has been removed in job titles in `Phantom Job List` for a more spacious layout.

### Auto Discard Items [General]

- New IPC `DailyRoutines.Modules.AutoDiscard.EnqueueByItems` coming with parameter `HashSet<uint> itemIDs` and return value `void`. A temporary list of items to be discarded could be passed in from outside.

### Auto Clear Fishing Spot Alert [General]

- The logic has been refactored.
- Global alert detection has been introduced. A notification would be sent if global alert has been triggered at a fishing spot.
- New configuration option `Logout After Global Alert`.
- Now `Instant Logout Or Exit` is a prerequisite module.

### Bait Switching Commands [Assistance]

- An issue that the module didn't work for `Swimbait` has been fixed.

### Fast Ride Pillion [General]

- The implementation has been adjusted, adopting a more agile approach.

### Auto Expert Delivery [UI_Operation]

- An issue that the first call to `Quickly Exchange Seals For Items` usually failed has been fixed.

### Auto Gather In The Diadem [Script]

- An issue that the character may be unable to land after windmire travelling has been fixed.

### Auto Item Desynthesis [UI_Operation]

- An issue that the option `Desynthesize entire stack.` were not enabled automatically has been fixed.

### Disable Background Game Rendering [System]

- An issue that the module didn't work has been fixed. The implementation has been refactored.

### Quick Chat Panel [UI_Optimization]

- The floating window has been fully refactored using vanilla assets instead of ImGui in the past.
- The configuration options `Window Height`, `Button Offset`, `Button Icon`, `Button Background` and `Font Scaling` have been removed.
- An issue of disordered `Special Icon Char` display has been fixed.
- An issue of being unable to send Auto-translation Dictionary has been fixed.
