## Framework Modifications

- An issue of being unable to access `The Firmament` in some automated processes has been fixed.

## Removed Modules

### Higher Quick Synthesis Cap [System]

- The module has been merged into the module `Better Crafting Log`.

## Module Modifications

### Instant Logout Or Exit [System]

- The module title has been adjusted. The previous title was `Instant Logout`.
- Now the module would work on all the following scenarios:
    - Clicking `Exit Game` in the title screen.
    - Ending the process via the title bar of the game window.
    - Clicking `Exit Game` in System Menu of the game.
    - Ending the process via the task bar.
- The module is now a `recommended module`.

### Auto Block Unnecessary Pop-ups [UI_Optimization]

- The blocking effect has been improved.
- More unnecessary pop-ups have been blocked.

### Auto Confirm Materia Melding And Retrieval [General]

- The melding efficiency has been improved.

### Auto Check Item Level [Combat]

- An issue that the module would work in Duty Recorder has been fixed.

### Better Reward Interface [UI_Optimization]

- The interface appearance and interaction animations have been fully refactored.

![OptimizedFreeShop-Addon](/assets/Changelog/2.0.6.0/OptimizedFreeShop-Addon.png)

### Better World Visit [System]

- Now all the servers within a data center are sorted lexicographically.

### Instant Character Acceleration [System]

- The logic has been optimized and the compatibility has been improved.

### Auto Confirm Quest Accepting [UI_Operation]

- An issue that the module didn't work has been fixed.
- The interrupt hotkey would no longer work on the module.

### Auto Skip Cutscenes [System]

- Adaption work for Patch 7.5 has been introduced.
- Now the `inspection` and `dart shooting` scenarios could be automated or skipped.
- More backup logic has been introduced to make sure the module work in all the applicable scenarios.

### Better Market Board [UI_Optimization]

- New UI component has been introduced, indicating the top three servers with the highest prices for the item and also the top three with the lowest prices.
    - To access the full detailed price information across all the servers and data centers, click `Show All` in the upper right corner.

![BetterMarketBoard-Overlay](/assets/Changelog/2.0.6.0/BetterMarketBoard-Overlay.png)

- The `HQ`-only filter could be enabled on the right.
- An issue that some unexpected behaviours regarding `HQ` items in Universalis mode has been fixed.
- Now the status of the `HQ`-only filter would persist regardless of the current item.
