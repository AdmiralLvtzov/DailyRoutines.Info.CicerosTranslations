## Framework Modifications

- A set of fuzzy search algorithms has been introduced for module searching.

## New Modules

### Item Using Command [Assistance]

*Recommended Module*

Add a new command /pdr {0} <item ID/item name/item name in pinyin> to use a specific item in the inventory.

![UseItemCommand](/assets/Changelog/2.0.7.0/UseItemCommand.png)

## Module Modifications

### Healer Helper [Action]

- The job icon of the target would be displayed in the notification of action redirection.

![HealerHelper-Notification](/assets/Changelog/2.0.7.0/HealerHelper-Notification.png)

### Auto Skip Cutscenes [System]

- The animation after logging in at Inn could be skipped now.

### Quickly Exchange Seals For Items [UI_Operation]

- An issue that, the logic could still be executed when the quantity was not set or was set to 0, has been fixed.

### Better Duty Finder Settings [UI_Optimization]

- An issue that the button order may vary has been fixed.

### Better Market Board [UI_Optimization]

- The efficiency of acquiring market data in the Item Help interface has been greatly improved.

### Auto Block Status Effect [Combat]

- An issue that the module didn't work has been fixed.
- An issue that, the character orientation would be fixed after successfully blocking any status, has been fixed.
- The implementation has been streamlined.
