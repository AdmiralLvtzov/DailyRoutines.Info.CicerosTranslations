## 本体修改

- 为模块搜索引入了一整套模糊搜索算法
- 手动为插件内各服务器数据排除了未实际开放的“亚马乌罗提”服务器

## 新增模块

### 物品使用指令 [辅助]

*推荐模块*

添加 /pdr item <物品 ID / 物品名称 / 物品名称拼音> 指令, 以快捷使用背包中指定物品

![UseItemCommand](/assets/Changelog/2.0.7.0/UseItemCommand.png)

### 快捷进出旅馆 [系统]

*推荐模块*

自动跳过进入/离开旅馆时的确认, 并提供 /pdr inn 指令以在副本外任一区域的任一位置一键进入旅馆

![FastEnterAndLeaveInn](/assets/Changelog/2.0.7.0/FastEnterAndLeaveInn.png)

### 更好的游戏内名称排序 [系统]

*推荐模块*

*仅国服* / *仅繁中服*

将多个游戏系统 (情感动作、无人岛、成就等) 中的项目名称排序方式, 由按 Unicode 码点排序改为按汉语拼音排序, 使界面显示更直观

![BetterNameSort](/assets/Changelog/2.0.7.0/BetterNameSort.png)

:::img-comp
![BetterNameSort-Before](/assets/Changelog/2.0.7.0/BetterNameSort-Before.png)
![BetterNameSort-After](/assets/Changelog/2.0.7.0/BetterNameSort-After.png)
:::

### 自动试穿全套外观 [界面操作]

*推荐模块*

为玩家检视界面新增按钮, 以一键试穿当前检视玩家全套实际外观装备

![AutoTryOnPlayerOutfit](/assets/Changelog/2.0.7.0/AutoTryOnPlayerOutfit.png)

![AutoTryOnPlayerOutfit-UI](/assets/Changelog/2.0.7.0/AutoTryOnPlayerOutfit-UI.png)

### 自动筹备稀有品 [界面操作]

- 新增了 `跳过绝境战相关装备` 配置项, 启用后将自动跳过可以由各绝境战副本图腾交换获得的装备, 默认启用

### 制作笔记界面优化 [界面优化]

- 新增配置项 `提醒简易制作完成`, 启用后, 当简易制作完成时, 自动发送通知信息提醒, 默认启用
- 调整了功能 `突破简易制作上限` 行为逻辑
    - 原先传入上限最多为 255 次
    - 现在允许直接在简易制作界面直接传入大于 255 的值 (上限依据所持材料数量而定), 并在多于 255 次时, 自动分批进行简易制作
- 在 `配方预览` 界面新增 `执行 X 次` 按钮, 以按照当前求解出来的配方步骤自动执行指定次数的制作
