## 本体修改

- 为模块搜索引入了一整套模糊搜索算法
- 手动为插件内各服务器数据排除了未实际开放的“亚马乌罗提”服务器

## 新增模块

### 物品使用指令 [辅助]

*推荐模块*

添加 /pdr item <物品 ID / 物品名称 / 物品名称拼音> 指令, 以快捷使用背包中指定物品

![UseItemCommand](/assets/Changelog/2.0.7.0/UseItemCommand.png)

### 快捷进出旅馆 [系统]

*推荐模块*

自动跳过进入/离开旅馆时的确认, 并提供 /pdr inn 指令以在副本外任一区域的任一位置一键进入旅馆

![FastEnterAndLeaveInn](/assets/Changelog/2.0.7.0/FastEnterAndLeaveInn.png)

### 更好的游戏内名称排序 [系统]

*推荐模块*

*仅国服* / *仅繁中服*

将多个游戏系统 (情感动作、无人岛、成就等) 中的项目名称排序方式, 由按 Unicode 码点排序改为按汉语拼音排序, 使界面显示更直观

![BetterNameSort](/assets/Changelog/2.0.7.0/BetterNameSort.png)

![BetterNameSort-Before](/assets/Changelog/2.0.7.0/BetterNameSort-Before.png)

- 支持跳过在旅馆内登录游戏时播放的动画效果

### 快捷军票交换所交换物品 [界面操作]

- 修复了未设置交换物品或交换数量设置为 0 时仍可以执行实际逻辑的问题

### 任务搜索器设置优化 [界面优化]

- 修复了界面按钮显示顺序会发生变更的问题

### 跨大区队员招募 [界面优化]

- 底层跟随服务迁移至 V2
- 支持包括驯兽师在内的更多版本新内容

### 更好的市场布告板 [界面优化]

- 大幅提升了道具帮助界面中市场数据的获取速度

### 自动屏蔽状态效果 [战斗]

- 修复了模块功能失效的问题
- 修复了在屏蔽任一状态效果后, 人物面向被意外锁定的问题
- 精简了模块实现

### 自动取回家具 [界面操作]

- 完全重构了模块
    - 使用游戏原生界面取代了 ImGui 绘制的悬浮窗

### 自动醒梦 [技能]

- 完全重构了模块逻辑
- 修复了 `绘灵法师` 无法自动使用 `醒梦` 的问题
- 修复了意外存在不需要使用 `醒梦` 的职业也进入了判断流程的问题
- 新增配置项 `发送聊天信息`, 默认禁用
- 调整了通知信息外观
- 调整了模块描述

### 自动恶名精英扫描器 [脚本]

- 重构了模块逻辑

### 自动云冠群岛采集 [脚本]

- 完全重构了模块逻辑, 采用了更加可维护且精简的架构

### 后台禁止渲染游戏画面 [系统]

- 优化了与 NamePlate 界面的兼容性

### 自动重定向地面目标类技能 [技能]

- 修复了首次发生技能重定向时, 可能造成的游戏卡顿问题

### 快捷跨界传送 [系统]

- 修改了服务器信息栏条目交互逻辑

### 快捷聊天面板 [界面优化]

- 优化了界面稳定性

### 更好的随机坐骑 [系统]

- 修复了已选坐骑展示模式与搜索模式混用的问题
