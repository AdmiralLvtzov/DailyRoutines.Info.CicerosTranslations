## 本体変更

- フランス語の完全なローカライズ対応を追加
- ファジー検索アルゴリズムを全面的に再構築
- メイン画面のカードコンテナの外観デザインとアニメーションを調整
- 開発画面で各作者のアバターがレンダリングされない問題を修正
- 作者の GitHub にジャンプする際に誤ったリンクが開かれる問題を修正
- モジュール構成の自動マイグレーション機能を追加
- 各モジュールのネイティブ UI におけるリソース管理とメモリ安全性のロジックを最適化
- 周囲プレイヤーの取得・更新のパフォーマンスと妥当性チェックロジックを最適化
- メンテナンス頻度が高すぎるため、`本日のオススメ` 機能を削除
- モジュールのリソース管理機構を最適化
- メイン画面の外観スタイルを最適化
    - 左サイドバーの垂直スクロールを、下部の操作可能なボタンにのみ適用
    - 左サイドバーからプラグイン名を削除し、プラグインアイコンのみ表示に変更
    - ウィンドウタイトルバーにあった DAU データを左サイドバーに移動
    - 右側コンテンツ領域のカードが左右の境界でサイドバーの折りたたみ/展開ボタンやスクロールバーに重なる問題を修正
    - プラグインに対応するゲームバージョンアイコンをウォーターマークとして右側コンテンツ領域に追加
- ログインしていない状態で認証トークンをコピーしても送信成功通知が表示される問題を修正

## 削除モジュール

### 自動コスモエクスプローラーレベリング [スクリプト]

### ミラプリ一括管理 [UI最適化]

## モジュール変更

### テレポUI最適化 [UI最適化]

- 検索判定ロジックを再構築
- 設定項目 `開いたら自動で検索欄にフォーカス` と `フォーカスを失ったら自動で閉じる` を追加（デフォルトで有効）

### ミラージュドレッサーを自動管理する [UI操作]

- `愛蔵品キャビネットに収納可能な装備を取り出す` 機能が、セットとして保存済みで収納可能な装備も取り出せるように対応

### パーティ募集フィルター画面の改善 [募集]

- モジュール名、モジュール説明、モジュール内部名を変更。再有効化が必要な場合があります
    - 旧名称 `募集フィルター`
- ゲーム内のパーティ募集関連設定の変更に対応し、ゲームネイティブの各種募集設定画面へのクイックアクセスをサポート
- モジュールオーバーレイを完全に再構築。ゲームネイティブ UI で描画する方式に変更し、`パーティ募集` 画面内に追加されたボタンから開閉するように改善

### 自動的に釣り場の警戒を解除する [一般]

- 設定項目 `自動キャスティング` を削除

### 不要なポップアップウィンドウを自動的に隠す [UI最適化]

- 非表示ロジックを最適化

### PT募集内容編集の情報を自動保存 [募集]

- モジュールロジックを完全に再構築
    - モジュール内部名を変更。再有効化が必要な場合があります
    - ゲームネイティブ UI でモジュール画面を完全に再構築
    - `自由コメント` 欄に入力した定型文の保存に対応

    - パーティ構成を除くすべての募集情報設定の保存に対応
- `おすすめモジュール` に昇格
- データ構造が大幅に変更されたため、自動データマイグレーションが機能しない可能性があります。データ移行に失敗した場合は、モジュール設定フォルダ内の `PartyFinderSettingRecord.json` ファイルをご確認ください。

### 自動ルーシッドドリーム [スキル]

- スキル使用タイミングの判定ロジックを最適化

### 自動的にF.A.T.E.を開始する [バトル]

- モジュール実装ロジックを調整

### ライブ配信匿名モード [一般]

- `ランダムNPC名` 匿名化モードにおける NPC 名のソースを調整
- パフォーマンス向上のため、チャット情報イベントの登録とチャット履歴からのプレイヤー名読み取りを削除
- `BannerParty` 画面の匿名化ロジックを再構築
- `NamePlate` 画面の匿名化ロジックを調整。プレイヤーは煩雑な情報を避けパフォーマンスを向上させるため、常にジョブアイコンのみを表示するように変更
- ジョブ名テキストの匿名化処理を除外
- メモリ安全性に関するロジックを最適化
- 認識精度を向上させ、無関係なテキストの誤処理を低減
- モデル匿名化機能を削除。必要な場合は `自動的にキャラクターの種族・性別を変更する` モジュールをご利用ください
- 設定項目 `ジョブ名を接頭辞として使用` を削除

### チャット自動応答 [一般]

- チャットテスト画面がストリーミング出力に対応
- ツール (Tool) 呼び出しに対応
    - 従来 `ワールドブック` に含めていた毎回の会話コンテキスト内の全ゲーム情報を、AI がツールを呼び出して取得する方式に変更。これに伴い設定項目 `ゲームコンテキストを有効にする` を削除
    - 従来は AI モデルが応答を生成するとそのままゲーム内チャットに使用されていましたが、ゲーム内チャットを `ツール` に変更し、さまざまなモデルに適応できるよう改善
- モデルコンテキスト長の設定に対応（デフォルト 25 万）。コンテキスト超過時は自動的に圧縮を実行し、従来の固定メッセージ数での圧縮を置き換え
- 履歴データの保存・呼び出しロジックを再構築
- 事前安全フィルターシステムのロジックを再構築し、カスタムブラックリストワードに基づくハードガード機構を追加
- チャットテスト画面の外観スタイルを最適化

### フレンドリストの改善 [UI最適化]

- リソース管理とメモリ安全性のロジックを最適化
- モジュールロジックを大幅に簡略化

### 即ログアウト・ゲーム終了 [システム]

- 処理フローを再構築。`サーバーとの接続が切断されました` ダイアログを表示せず、直接ログイン前のタイトル画面に戻るように変更

### ログイン操作時のクールタイムロックを解除 [システム]

- ログイン関連のリクエスト間隔短縮に対応
    - キャラクターデータ取得リクエスト：
        - 変更前：2 秒、3 秒、5 秒、5 秒、5 秒、5 秒、7 回目以降は固定 30 秒
        - 変更後：常に固定 1 秒
    - キャラクターログインリクエスト：
        - 変更前：7 回目までは固定 5 秒間隔、7 回目以降は固定 30 秒
        - 変更後：常に固定 1 秒
    - キャラクターがログイン中のため強制ログアウトが必要な場合：
        - 変更前：固定 30 秒
        - 変更後：固定 1 秒
- 機能追加に伴い、モジュール説明を調整

### クイック・ワールド間テレポ [システム]

- `即ログアウト・ゲーム終了` モジュールの再構築に伴い、ロジックを追随して再構築

### コンテンツ突入準備の残り時間を表示 [バトル]

- モジュールロジックを完全に再構築し大幅に簡略化
    - ゲーム内部の正確なマッチングタイムスタンプを使用し、独自管理を廃止
    - 画面更新速度と正確性を大幅に向上

### コンテンツから自動退出する [バトル]

- 設定画面の描画ロジックを再構築
- 設定項目 `即時にコンテンツを退出する` を追加。ここで設定したコンテンツ完了時は遅延を無視して即座に退出しますが、`ブラックリストコンテンツ` より優先度は低くなります

### テレポUI最適化 [UI最適化]

- `/pdrtelepo` コマンドでエーテライト以外の目標を選択した際に判定エラーが発生する問題を修正

### ブラックリスト改修 [システム]

- UI テキストの表現を一部調整
- プレイヤーを `ローカルブラックリスト` に追加した際の操作フィードバックが不足していた問題を修正
- ブラックリストにプレイヤー検索機能を追加
- ログインしていない状態で `共有ブラックリスト` の `更新ログ` に表示されるプレイヤー詳細情報が読み込めない問題を修正
- プレイヤー情報の取得に成功した場合、ContentID を非表示にして画面を整理。マウスホバーやクリックコピーでアクセス可能

### 自動コスモフォーチュン [一般]

- `惑星アウクセシア` 内の `アウクセシアクレジット` に対応

### 周囲のプレイヤーの数を自動で表示する [一般]

- オーバーレイとサーバー情報欄のツールチップ表示スタイルを統一・最適化。各行の表示順序は左から順に：ジョブアイコン、プレイヤー名、プレイヤーのホームワールド名
- `蜃気楼の島 クレセントアイル` に加え、`セイブ・ザ・クイーン` と `禁断の地 エウレカ` 関連エリアでも `エリア人数` の表示に対応

### 自動ログイン [一般]

- 有効なキャラクターが存在しないサーバーで処理が停止する問題を修正。自動的に次の設定を試行するよう改善

### ID情報を自動表示 [UI最適化]

- `バフ・デバフ` の ID 表示と `ナビマップ` 画面の `天気` ID 表示を復元
- `ターゲット` の ID 表示更新ロジックを最適化

### 特殊フィールド探索・進入コマンド [補助]

- `惑星アウクセシア`（引数 `auxesia`）に対応
- 自動でマウントを降りる機能に対応
