## 본체 변경

- 완전한 프랑스어 로컬라이징 리소스 지원 추가
- 퍼지 검색 알고리즘 시스템 전면 재구축
- 메인 화면 카드 컨테이너의 외관 디자인 및 애니메이션 조정
- 개발 화면에서 각 작성자 아바타가 렌더링되지 않는 문제 수정
- 작성자 GitHub 이동 시 잘못된 링크가 열리는 문제 수정
- 모듈 구성 자동 마이그레이션 로직 추가
- 각 모듈의 네이티브 UI 리소스 관리 및 메모리 안전 로직 최적화
- 주변 플레이어 획득 및 업데이트 성능과 적법성 검사 로직 최적화
- 유지보수 빈도가 너무 높아 `오늘의 추천` 기능 제거
- 모듈 리소스 관리 메커니즘 최적화
- 메인 화면 외관 스타일 최적화
    - 왼쪽 사이드바의 세로 스크롤이 하단 상호작용 버튼에만 적용되도록 변경
    - 왼쪽 사이드바에서 플러그인 이름 제거, 플러그인 아이콘만 표시
    - 기존 창 제목 표시줄의 일일 활성 사용자 데이터를 왼쪽 사이드바로 이동
    - 오른쪽 콘텐츠 영역 카드의 좌우 테두리가 사이드바 접기/펼치기 버튼 및 스크롤바와 겹치는 문제 수정
    - 플러그인에 대응하는 게임 버전 아이콘을 워터마크로 오른쪽 콘텐츠 영역에 추가
- 로그인하지 않은 상태에서 인증 토큰을 복사해도 전송 성공 알림이 표시되는 문제 수정

## 모듈 변경

### 텔레포 UI 최적화 [UI 개선]

- 검색 판정 로직 재구축
- 설정 항목 `열 때 자동으로 검색창 포커스` 및 `포커스 해제 시 자동 닫기` 추가, 기본 활성화

### 프로젝션 화면 자동 관리 [UI 조작]

- `추억의 보관함에 수납 가능한 장비 꺼내기` 기능이 세트로 저장되었지만 수납 가능한 장비도 꺼낼 수 있도록 지원

### 개선된 파티 찾기 필터 UI [모집]

- 모듈 이름, 모듈 설명 및 모듈 내부 이름 변경, 재활성화가 필요할 수 있음
    - 이전 이름 `모집 필터`
- 게임 내 파티 모집 관련 설정 변경을 지원하고, 게임 기본의 각종 모집 설정 화면에 빠르게 접근 가능
- 모듈 오버레이를 완전히 재구축, 게임 네이티브 UI로 렌더링하도록 변경하고, `파티 찾기` 화면 내에 추가된 버튼으로 열고 닫도록 개선

- 숨김 로직 최적화

### 파티 모집 내용 자동 저장 [모집]

    - 모듈 내부 이름 변경, 재활성화가 필요할 수 있음
    - 게임 네이티브 UI로 모듈 화면을 완전히 재구축
    - `소개말` 입력란에 입력한 상용구 저장 지원

    ![AutoRecordPartyFinderSettings-Addon](/assets/Changelog/2.0.8.0/AutoRecordPartyFinderSettings-Addon.png)

    - 저장된 모집 정보를 우클릭하여 메뉴를 열고 선택한 내용으로 현재 모집 정보 업데이트 지원
    - 파티 구성을 제외한 모든 모집 정보 설정 저장 지원
- `추천 모듈`로 승격
- 데이터 구조가 크게 변경되어 자동 데이터 마이그레이션이 작동하지 않을 수 있음. 데이터 이전에 실패한 경우 모듈 설정 폴더 내 `PartyFinderSettingRecord.json` 파일을 확인해 주세요.

### 자동 자각몽 [스킬]

- 스킬 사용 시점 판정 로직 최적화

### 자동 돌발 임무 [전투]

- 모듈 구현 로직 조정

### 라이브 방송 익명 모드 [일반]

- `랜덤 NPC 이름` 익명화 모드에서 NPC 이름 출처 조정
- 성능 향상을 위해 채팅 정보 이벤트 등록 및 채팅 기록에서의 플레이어명 읽기 제거
- `BannerParty` 화면의 익명화 로직 재구축
- `NamePlate` 화면의 익명화 로직 조정, 플레이어가 혼란 정보를 피하고 성능 향상을 위해 항상 직업 아이콘만 표시
- 직업명 텍스트의 익명화 처리 제외
- 메모리 안전 관련 로직 최적화
- 인식 정확도 향상, 관련 없는 텍스트의 오처리 감소
- 모델 익명화 기능 제거, 필요한 경우 `자동 캐릭터 종족/성별 변경` 모듈 사용
- 설정 항목 `직업명을 접두사로 사용` 제거

### 채팅 자동 응답 [일반]

- 채팅 테스트 페이지에서 스트리밍 출력 지원
- 도구(Tool) 호출 지원
    - 기존 `월드북`에 포함되어 매 대화 컨텍스트에 포함되던 전체 게임 정보를 AI가 도구를 호출하여 관련 정보를 얻도록 변경. 이에 따라 설정 항목 `게임 컨텍스트 활성화` 제거
    - 기존에는 AI 모델이 응답을 생성하면 그대로 게임 내 채팅 응답으로 사용되었으나, 이제 게임 내 채팅 응답을 `도구`로 변경하여 다양한 모델에 더 잘 대응
- 모델 컨텍스트 길이 설정 지원, 기본값 25만, 컨텍스트 초과 시 자동으로 컨텍스트 압축 실행, 기존 고정 메시지 수 압축 방식 대체
- 기록 데이터 저장 및 호출 로직 재구축
- 사전 안전 필터 시스템 로직 재구축, 사용자 정의 블랙리스트 단어 기반 하드 가드 메커니즘 추가
- 채팅 테스트 화면 외관 스타일 최적화

### 친구 목록 개선 [UI 개선]

- 리소스 관리 및 메모리 안전 로직 최적화
- 모듈 로직 대폭 간소화

### 즉시 로그아웃 / 종료 [시스템]

- 처리 흐름 재구축, `서버와 접속이 끊어졌습니다` 대화상자를 표시하지 않고 직접 로그인 전 타이틀 화면으로 돌아가도록 변경

### 로그인 조작시 쿨다운 잠금 해제 [시스템]

- 로그인 관련 요청 간격 단축 지원
    - 캐릭터 데이터 획득 요청:
        - 변경 전: 2초, 3초, 5초, 5초, 5초, 5초, 7회차부터 고정 30초
        - 변경 후: 항상 고정 1초
    - 캐릭터 로그인 요청:
        - 변경 전: 7회차까지 고정 5초 간격, 7회차부터 고정 30초
        - 변경 후: 항상 고정 1초
    - 캐릭터가 접속 중인 상태로 인해 강제 접속 종료가 필요한 경우:
        - 변경 전: 고정 30초
        - 변경 후: 고정 1초
- 기능 추가에 따라 모듈 설명 조정

### 빠른 서버 텔레포트 명령어 [시스템]

- `즉시 로그아웃 / 종료` 모듈의 재구축에 따라 로직을 함께 재구축

### 콘텐츠 입장 준비 남은 시간 표시 [전투]

- 모듈 로직을 완전히 재구축하고 대폭 간소화
    - 게임 내부의 정확한 매칭 타임스탬프 사용, 자체 관리 폐기
    - 화면 업데이트 속도 및 정확성 대폭 향상

### 컨텐츠 자동 퇴장 [전투]

- 설정 화면 그리기 로직 재구축
- 설정 항목 `즉시 컨텐츠 퇴장` 추가. 여기서 설정한 컨텐츠 완료 시 지연을 무시하고 즉시 퇴장하지만, `블랙리스트 컨텐츠`보다 우선순위가 낮음

### 텔레포 UI 최적화 [UI 개선]

- `/pdrtelepo` 명령어가 에테르라이트 이외의 목표를 선택할 때 판정 오류가 발생하는 문제 수정

### 차단 목록 개편 [시스템]

- UI 텍스트 일부 표현 조정
- 플레이어를 `개인 블랙리스트`에 추가할 때 조작 피드백이 부족했던 문제 수정
- 차단 목록에 플레이어 검색 기능 추가
- 로그인하지 않은 상태에서 `공유 블랙리스트`의 `업데이트 로그` 내 플레이어 상세 정보를 불러올 수 없는 문제 수정
- 플레이어 정보 획득 성공 시 ContentID를 숨겨 인터페이스 정리, 마우스 호버 또는 클릭 복사로 접근 가능

### 자동 우주 복권 [일반]

- `행성 아욱세시아` 내 `아욱세시아 화폐` 지원

### 주변 플레이어 자동 표시 [일반]

- 오버레이와 서버 정보 표시줄 도구 설명의 표시 스타일 통일 및 최적화, 각 행은 왼쪽부터 직업 아이콘, 플레이어명, 플레이어의 고향 서버명 순서로 표시
- `초승달 섬` 외에도 `세이브 더 퀸` 및 `금단의 땅 에우레카` 관련 영역에서 `지역 인원수` 표시 지원

### 자동 로그인 [일반]

- 유효한 캐릭터가 없는 서버에서 처리 중단 문제 수정, 자동으로 다음 설정 시도

### ID 정보 자동 표시 [UI 개선]

- `강화 및 약화 상태` ID 표시와 `미니맵` 화면의 `날씨` ID 표시 복원
- `대상` ID 표시 갱신 로직 최적화

### 특수 던전 탐색 진입 명령어 [보조]

- `행성 아욱세시아` (인자 `auxesia`) 지원
- 자동으로 탈것 하차 지원
