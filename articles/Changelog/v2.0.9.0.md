## 本体修改

- 重构了托盘通知触发链路
- 优化了模块管理逻辑
- 修复了设置界面卡片组件绘制可能超出右侧边界的问题
- 调整了模块分类 (以下内容与普通用户无关, 仅为内部分类标准调整声明)
    - 删除分类:
        - `界面操作` (`UIOperation`)
        - `界面优化` (`UIOptimization`)
    - 新增分类:
        - `界面` (`Interface`)
    - 调整分类内部名称:
        - `通知`: `Notice` → `Notification`
    - 划分为 `一等分类` 与 `二等分类`, `一等分类` 较为笼统, `二等分类` 较为具体。若一个模块可被同时归入 `一等分类` 或 `二等分类`, 则会被分类为对应的 `二等分类`。若一个模块同时可被归入多个 `一等分类` 或 `二等分类`, 则优先分入显示排序更靠后的分类 (具体见下)
    - `一等分类` 包含:
        - 一般
        - 通知
        - 系统
        - 界面
        - 战斗
        - 脚本
        - 辅助
    - `二等分类` 包含:
        - 副本
        - 招募
        - 技能
        - 生产采集
        - 金碟
    - 原 `界面操作` 与 `界面优化` 模块分类统一转为 `界面`。其余模块分类调整详见下方 `模块分类调整`
- 重构了插件生命周期, 全面改造为异步初始化与异步卸载
- 修复了在主界面开启状态下卸载插件可能会导致空引用异常被扔出的问题
- 修复了因卸载插件时仍在获取周边玩家信息导致空引用异常被扔出的问题
- 大幅优化了插件内部的网络连接逻辑
- 修复了在认证节点均不可用时, 加载插件会导致游戏进入可恢复的无响应状态的问题

## 模块名称调整

- `自动显示 ID 信息`: 由于内部名称存在拼写错误, 调整后内部名称为 `AutoDisplayIDInformation`, 需要重新启用并配置该模块
- `自动进退副本一次`: 修改名称为 `进退副本指令`, 调整后内部名称为 `JoinExitDutyCommand`, 需要重新启用并配置该模块
- `快速重置木人仇恨`: 调整了内部名称, 调整后内部名称为 `FastResetStrikingDummy`, 需要重新启用并配置该模块
- `目标标记调整`: 调整名称为 `自动调整目标标记`, 调整内部名称为 `AutoAdjustNamePlateIcon`, 需要重新启用并配置该模块
- `小队列表显示目标标记`: 调整名称为 `自动小队列表显示目标标记`, 调整内部名称为 `AutoDisplayMarkerInPartyList`, 需要重新启用并配置该模块
- `多目标追踪器`: 调整名称为 `自动追踪玩家`, 调整内部名称为 `AutoTrackPlayers`, 需要重新启用并配置该模块

## 模块修改

### 自动追踪玩家 [系统]

- 完全重构了模块逻辑
- 移除了 `永久追踪` 功能与历史记录功能, 现在仅保留原 `临时追踪` 功能, 在区域切换时清空
- 新增游戏原生界面, 可以分配每个玩家对应的目标标记, 并可在界面内执行移除、选中等操作

![AutoTrackPlayers-Addon](/assets/Changelog/2.0.9.0/AutoTrackPlayers-Addon.png)

### 自动调整目标标记 [战斗]

- 修改为推荐模块
- 重构了模块实现逻辑

### 进退副本指令 [辅助]

- 移除了在线验证要求
- 修改为推荐模块

### 快速重置木人仇恨 [战斗]

- 修改为推荐模块

### 自动取消登录操作冷却锁 [系统]

- 优化了排队时的处理逻辑, 避免对话框界面反复出现消失

### 自动登录 [一般]

- 修改模式 `登陆一次` 的逻辑为: 当前进程只登录一次, 不再受模块生命周期影响
- 重构了配置, 改为使用 "服务器+角色名" 的匹配模式, 需要重新配置
    - `服务器` 语义修改, 原先为角色索引对应的当前服务器, 现在改为角色名称对应的原始服务器
- 因应配置调整, 一并修改了命令传参, 可能需要调整已有的命令
    - 原仅支持传入玩家角色索引, 新增支持传入玩家角色名称
    - 固定首个参数为 `角色索引/角色名称`
    - 固定第二个参数为可选的 `服务器名称 / 服务器 ID`
- 内部逻辑完全重构, 不再遍历服务器信息与做多次重试尝试, 而是一次性登入最佳可登录角色, 并支持登录至正在超域旅行的角色

### 快捷跨界传送 [系统]

- 在执行登出时, 会确保 `自动登录` 模块相关逻辑关闭后再执行后续逻辑
- 在执行登出时, 会确保角色已登出后再执行后续逻辑

### 小队列表显示目标标记 [战斗]

- 修复了 `队伍列表` 界面未显示时, 仍会尝试修改界面节点的问题

### 钓饵切换指令 [辅助]

- 支持 `宇宙探索` 玩法中的钓饵

### 自动存入收藏柜 [界面]

- 修复了会尝试存入已在收藏柜中的物品的问题
- 极大提升了存入速度
- 修复了存入日志信息中无法显示物品名称的问题
- 悬浮窗改为使用游戏原生界面绘制
