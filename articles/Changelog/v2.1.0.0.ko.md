## 본체 변경

- 버전 업데이트 알림 화면의 외관 간소화
- 플러그인 로드 및 언로드 로직 최적화
- `로고 화면 건너뛰기` 기능이 초기화 순서로 인해 작동하지 않던 문제 수정
- 주변 플레이어 획득 시 플레이어 여부 판단 로직 강화
- 플러그인 영역 간 이동 기능에 자동 텔레포트선 사용 로직 추가

## 신규 모듈

### 시스템 메뉴 자동 간소화 [인터페이스]

*추천 모듈*

시스템 메뉴에서 일상적인 플레이에 불필요한 항목을 숨겨 UI를 간소화합니다.

![SimplifiedSystemMenu](/assets/Changelog/2.1.0.0/SimplifiedSystemMenu.png)

### 임무 내 귓속말 제한 자동 해제 [콘텐츠]

*추천 모듈*

임무 내에서 "귓속말"(/tell) 채널을 통해 메시지를 주고받을 수 없는 제한을 해제합니다.

### "스사노오 토벌전" 도우미 [콘텐츠]

*추천 모듈*

임무 "스사노오 토벌전"에 입장 후, 올바른 "신의 바위"를 빠르게 파괴할 수 있도록 "구속" 상태 효과를 지닌 파티원에게 자동으로 마커를 표시합니다.

### 시스템 사운드 효과음 최적화 [시스템]

*추천 모듈*

시스템 효과음 명령어(<se>)를 모든 채팅 채널에서 출력할 수 있게 합니다. 또한 음량 증폭을 위해 중복 사용할 때의 최대 3회 횟수 제한을 해제합니다.

### 적대 목록 개선 [전투]

- 모듈 구현 로직 간소화, 전체 성능 향상
- 새 설정 항목 `타겟팅 제한 자동 해제`. 일부 콘텐츠에서는 플레이어가 일정 시간 동안 "적대 목록"에서 특정 적대 대상을 선택할 수 없게 됩니다. 예: 콘텐츠 「조트 탑」의 보스 2 「도그」가 「마누샤 혼란」 스킬 시전 중, 이 설정을 활성화하면 자동으로 처리하여 선택 가능한 적을 항상 선택할 수 있게 합니다. 기본값 활성화
- 모듈 내 비관리 리소스의 생명주기 관리 최적화
- 추천 모듈로 `숨겨진 시전 바 자동 표시` 추가. 모듈 내 숨겨진 시전 정보 표시 기능 제거
- 외관 스타일 재설계
    - 적개심 수치 표시 제거. 서버는 대부분 자신의 적개심 수치만 전송하므로 정확한 개인 적개심 수치를 아는 의미가 적음
    - HP 백분율 수치 표시 가중치 향상
    - 사용자 정의 텍스트 형식과 배경 불투명도 설정 제거
    - HP 백분율 수치는 왼쪽에, 상태 효과는 오른쪽에 고정 표시

    ![OptimizedEnemyList-Interface](/assets/Changelog/2.1.0.0/OptimizedEnemyList-Interface.png)

### 텔레포 UI 최적화 [인터페이스]

- 스마트 검색 기억 시스템 도입. 매번의 검색어와 선택을 기억하여 더욱 편리한 조작감 제공

### 시스템 사운드 재생 빈도 및 음량 제한 [시스템]

- `음량 강도` 설정 항목 제거. 모든 시스템 효과음의 음량 강도에 대한 스로틀 지원

### 자동 돌발 임무 [스크립트]

- 모듈 내부 로직 완전 재구축. 더욱 간결하고 효율적으로 개선
- 로그 정보 출력 지원
- 전투 중 적대 대상 탐색 접근 로직 제거. BossMod (Reborn)에 완전 위임
- 수집형 돌발 임무에서 전투보다 수집 가능한 오브젝트를 우선 탐색
- 새로운 충돌 모듈 `스킬 대상의 차폐 상태 무시` 추가
- 추천 모듈 `버디(초코보) 자동소환` 추가
- 두 빛깔 보석 교환 중에는 경로 탐색을 통해 해당 위치로 이동하도록 변경
- 「희망 잃은 소녀」/「희망 잃은 자」 예상 처치 시간 감지 추가. 처치 시간이 돌발 임무 잔여 시간보다 길면 무시
- 돌발 임무 종료 후에도 전투 상태가 지속될 경우, 콘텐츠 퇴장 시 자동으로 가장 가까운 안전 지대로 복귀
- 돌발 임무 대상보다 비-돌발 임무 대상의 우선순위가 높았던 문제 수정
- 설정 화면 그리기 로직 재구축
- 선행 플러그인/모듈 설정 설명 추가

### 거래 자동 취소 [시스템]

- 새 설정 항목 `지연`. 0이 아닌 값을 설정하면 거래 취소 전 지정된 시간 동안 대기합니다. 기본값 0
- 추가 텍스트 명령어가 `/wait` 명령어와 `<wait>` 매개변수 지원

### 자동 공격 자동 수행 [전투]
