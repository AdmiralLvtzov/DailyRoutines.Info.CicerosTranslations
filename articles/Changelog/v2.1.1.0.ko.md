## 모듈 변경

### 퀵 단골 거래 정보 화면 [인터페이스]

- 단골 `티소쟈` 정보 추가
- 게임 네이티브 우클릭 메뉴 UI로 기존 ImGui UI 대체
- 리소스 정리 메커니즘으로 인한 게임 충돌 가능성 수정
- 단골 `카이 시르` 위치 확인 시 잘못된 지도가 표시되던 문제 수정
- `텔레포` 기능의 `테스트 코드` 요구사항 제거 [중국 서버만]

### 파티 모집 내용 자동 저장 [파티 모집]

- 우클릭 메뉴를 열 때 게임이 충돌할 수 있는 문제 수정

### 자동 돌발 임무 [스크립트]

- `솔루션 나인`으로 텔레포 후 이동 불가 지역에 갇히는 문제 수정
- 돌발 임무 내 경로 탐색 로직 최적화 및 간소화
- `희망 잃은 소녀` / `희망 잃은 자`가 현재 도달 불가능한 위치에 나타났지만 이미 전투 중일 때 흐름이 멈추는 문제 수정
- 현재 돌발 임무 대상이 전혀 생성되지 않았을 때 자동으로 돌발 임무 중심지로 이동하지 않던 문제 수정
- 돌발 임무 내에서 자동으로 `전력 질주` 사용
- 선행 모듈 `자동 마테리아 정제`, `자동 돌발 임무 레벨 동기화`, `돌발 임무 자동 시작` 제거
- 자동 레벨 동기화 관련 로직 추가
- 수동 시작이 필요한 돌발 임무로 향할 때 상태 머신 순환에 반복적으로 빠지는 문제 수정
- 수동 시작이 필요한 돌발 임무로 향할 때, 먼저 시작 NPC까지 경로 탐색 후 탈것 해제·대상 선택 후 돌발 임무를 시작하여 보다 일반 플레이어 행동에 가깝게 조정
- NPC와의 상호작용 범위 임계값 조정
- 수집형 돌발 임무 도착 직후, 적대 대상이 아닌 수집 가능 오브젝트로 자동 경로 탐색
- 수집형 돌발 임무에서 각 제출 아이템이 얼마나 많은 돌발 임무 진행도로 변환되는지 지능적으로 감지하고, 진행도에 따라 제출 시점을 선택

### 텔레포 UI 최적화 [인터페이스]

- 모듈 로직 완전 재구축, 각종 특수 에테라이트 기능 조작 완전 지원
- `우주 개척` 현장 내 에테라이트로 텔레포 지원
- 동일 구역의 다른 에테라이트로 텔레포할 때 정상적인 텔레포 흐름에 진입하지 않던 문제 수정
- 전체 화면에서 텔레포 실행 후 창이 자동으로 닫히지 않던 문제 수정
- 알림 메시지 외관 스타일 변경
- 메인 에테라이트가 무료/즐겨찾기 지점일 때 다른 도시 내 에테라이트도 할인된 가격으로 표시되던 문제 수정

### 자동 수리 [시스템]

- 동작 로그 표시
- 알림 메시지 외관 스타일 조정
- 모듈 로직 최적화 및 간소화
- 극히 드물게 알림 메시지가 중복 전송되던 문제 수정

### 낚시 상태 자동 유지 [시스템]

- 모듈 설명이 불완전했던 문제 수정

### 버프 중복 방지 [액션]

- 극히 드물게 발생할 수 있는 null 참조 예외 수정

### 자동 자각몽 [액션]

- 일부 상황에서 채팅 알림 메시지가 중복 전송되던 문제 수정
- 알림 메시지 외관 스타일 조정

### 적대 목록 개선 [전투]

- UI 갱신 로직 최적화
- 시전 텍스트 위치 조정
- 150% 이외의 UI 배율에서 상태 효과가 다른 요소와 어긋나는 문제 수정

### 자동 마테리아 정제 [시스템]

- 동작 로그 표시
- 알림 메시지 외관 스타일 조정
