# 自动精选 / AutoAetherialReduction

## IsBusy

检查当前模块是否正忙

- 名称: `DailyRoutines.Modules.AutoAetherialReduction.IsBusy`
- 返回值 `bool`
  - `true`: 是
  - `false`: 否 / 模块未启用

## StartReduction

开始精选

- 名称: `DailyRoutines.Modules.AutoAetherialReduction.StartReduction`
- 返回值 `bool`
    - `true`: 发送开始精选请求成功
    - `false`: 模块未启用
