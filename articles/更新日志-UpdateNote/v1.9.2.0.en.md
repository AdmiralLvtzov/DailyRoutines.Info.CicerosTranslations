## Modifications on Daily Routines

- The permission system was refactored. Unexpected behaviours of modules will not be allowed therefore no longer jeopardize the vanilla game running.
- FFXIVClientStucts APIs in some modules were updated. Therefore some modules may no longer work on older Dalamud versions.
- Some dependencies were updated, therefore the stability was improved.
- Some FFXIVClientStructs updates were introduced. Some code was streamlined.
- The module loading mechanism was optimized.
- The appearance and animation of settings were remastered.

![Settings-UI](/assets/Changelog/1.9.2.0/Settings-UI.png)

- The appearance and animation of module configurations were remastered.
- The appearance and animation of search bars were remastered. 
- The appearance and animation of the About interface were remastered.
- The appearance and animation of the Authentication interface were remastered.
- The notification appearance was optimized. A plugin icon was introduced to improve recognizability.
- An issue that layouts and fonts may not be applied as expected has been fixed.
- Now the screen resolution will be considered while scaling interfaces.

## Module modifications

### Better Crafting Log [UI_Optimization]

- The appearance while exchanging items for currency in `Available Deal` was optimized.
- An issue of broken item icons in `Available Deal` has been fixed.
- The appearance of buttons to switch among jobs in `Crafting Log` was optimized.
- Now predictions in `Crafting Log` are always hidden to prevent overlapping.

### Higher Quick Synthesis Limit [System]

- An compatibility issue has been fixed.

### Better Market Board [UI_Optimization]

- An issue of `Sequence contains no element` exceptions while no item for sale has been fixed.
- The performance while drawing the window was improved.

### Occult Crescent Helper [Assistance]

- An issue that some elements regarding jobs and traits on the right side of `Phantom Job List` couldn't be interacted has been fixed.

### Better Friend List [UI_Optimization]

- An issue that some elements in the `Search Settings` interface were misplaced has been fixed.

### Auto Expert Delivery [UI_Operation]

- An issue that some elements in the module interface were misplaced has been fixed.

### Better Target Bar [UI_Optimization]

- An issue that cast bars under target names may not indicate cast progress properly while engaging multiple targets in some rare cases has been fixed.

### Better Teleportation [UI_Optimization]

- An issue of `Collection was modified` exceptions due to improper array destruction has been fixed.
- An issue that the module may get stuck after logging out and reloading the module has been fixed.
