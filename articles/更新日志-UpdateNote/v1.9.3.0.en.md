## New modules

## Module modifications

### Right-click Movement [General]

- An issue that the `vnavmesh` mode couldn't be selected has been fixed.
- The drawing logic of the configuration interface was refactored.
- The display of destination lines was optimized.
- The module will automatically optimize destination coordinate calculations using `vnavmesh` if enabled.

### Auto Leave Duty [Combat]

- The drawing logic of the configuration interface was refactored.
- Some misleading tips are not intercepted.
- An issue that the option `Force to leave the duty` didn't work has been fixed.

### Instantly Leave Duty [Combat]

- Some misleading tips are not intercepted.

### Better Teleportation [UI_Optimization]

- Brand new interaction animations within the floating window were introduced.
- The layout logic within the floating window was refactored.
- Brand new display animations within the floating window were introduced.
- Now all the aetherytes and aethernet shards in the current zone will be displayed on the floating map.
- An issue that the positions of aetherytes and aethernet shards may be shifted a bit has been fixed.
- Now users could scale the floating map by pressing the interrupt hotkey and scrolling.
- Now users could pin the floating map by holding the interrupt hotkey, in order to interact with the aetherytes and aethernet shards on it.
