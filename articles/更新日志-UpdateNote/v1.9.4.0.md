## 本体修改

- 适配了 Dalamud API 14 与 7.4 版本
- 新增配置项 `默认启用推荐模块`, 启用后, 当本地首次初始化任一 `推荐模块` 时, 将其设置为启用状态, 默认启用
    - `推荐模块`: 由开发组筛选，能改善大多数玩家体验, 且不会过多改变日常游玩方式的模块

## 新增模块

### 快捷天气时间设置 [界面优化]

提供游戏原生风格的快捷天气时间设置界面, 并允许保存为区域设置自动应用, 通过交互导向地图界面天气或使用 /pdr wt 打开

## 模块信息调整

- 名称: `快捷副本区切换指令` → `快捷副本区切换`
- 内部名称: `InstanceZoneChangeCommand` → `FastInstanceZoneChange`
- 描述: `快速切换副本区并显示副本区信息, 支持自动重试、自动上下坐骑与自动传送等, 通过 /pdr wt 指令快捷切换或点击服务器信息栏条目打开界面交互切换`

<br/>

- 名称: `快捷跨界传送指令` → `快捷跨界传送`
- 内部名称: `WorldTravelCommand` → `FastWorldTravel`
- 描述: 
```
快速跨界传送并显示服务器信息, 支持三主城内任意位置发起跨界、副本外任意区域开始自动跨服等, 通过 /pdr worldtravel 指令快捷跨界传送或点击服务器信息栏条目打开游戏原生风格界面交互跨界
支持快捷超域旅行并实时显示各服务器超域旅行拥挤度 [国服特供]
```
