## Modifications on Daily Routines

- An issue that some texts may be broken after applying a new font or adjusting the font size has been fixed.
- An issue of resource leaks related to fonts has been fixed.
- Now the default Dalamud font could be applied.
- An issue of resource leaks related to recommended modules has been fixed.
- An issue that a module may be reloaded multiple times unexpectedly after certain changes has been fixed.
- The localization performance was improved.
- The logic of auto command completion was adjusted. The left-click logic remains. The right-click logics is to send the command directly.
- An issue that Chinese number format may appear unexpectedly in some cases has been fixed.
- An issue that module reloading tasks may be aborted earlier than expected has been fixed.
- An issue of delayed memory free-up due to unexpected vanilla UI element uninstallations has been fixed.
- An exception due to online data acquisitions initiated by unreliable threads has been fixed.
- The initialization performance was improved.
- The performance of traversing in-game object database was improved.
