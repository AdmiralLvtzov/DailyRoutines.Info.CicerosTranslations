## Modifications on Daily Routines

- A game performance issue due to delayed execution of the task queue has been fixed.
- A internal thread management which is overly complicated was removed for better performance and stability.
- The performance while requesting data from Universalis was improved.
- The local time generation is now more accurate.

## Module modifications

### Better Company Chest [UI_Optimization]

- A text issue has been fixed.

### Auto Collect Subaquatic Voyages [UI_Operation]

- The vanilla error prompt `Unable to access airship/submersible.` would no longer present, since it's irrelevant and may cause confusion.

### Auto Mark Aether Currents [UI_Optimization]

- An `OutOfRange` exception may occur in non-instanced areas has been fixed.

### Auto Start FATE [Combat]

- The performance was improved.

### Auto Cutscene End Notification [Notification]

- The module was completely refactored adopting a more efficient and reliable implementation.

### Fast Teleportation Panel [UI_Optimization]

- A more accurate logic for identifying quest NPC was introduced for quest teleportation.
- An issue that the actual destination may deviate from the expected in some rare cases has been fixed.

### Auto Use Quest Event Item [General]

- The performance was greatly improved.
- A retry logic was introduced.

### Better Instance Switching [System]

- The performance was improved. The data update delay was decreased.

### Better World Visit [System]

- The performance was improved. The data update delay was decreased.

### Occult Crescent Helper [Assistance]

- The performance was improved.

### Better Enmity List [UI_Optimization]

- The performance while updating data was improved.

### Alexander Cuff Of Father Helper [Assistance]

- The module was refactored. The performance was greatly improved.

### Auto Open Treasure Coffers [Combat]

- The performance was improved.

### Better Auto Follow [System]

- The performance was greatly improved.

### Frontline Map Radar [Combat]

- The performance was greatly improved.

### Quickly Exchange Seals For Items [UI_Operation]

- A crash issue due to new data structures in the interface has been fixed.

### Hullbreaker Isle Helper [Assistance]

- The module was refactored. The performance was greatly improved.
