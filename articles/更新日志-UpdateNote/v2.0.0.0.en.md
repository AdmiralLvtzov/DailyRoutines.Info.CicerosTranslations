## Modifications on Daily Routines

- We're now having a brand new plugin icon!
- New layout style `Tropical Sanctuary`.
- Various animations in the interface were optimized.
- The Text-To-Speech feature was refactored.
    - All the available narrators from Microsoft at the moment are all supported in Daily Routines now.
    - An issue of unexpected cache after tweaking `Volume` has been fixed.
- A new implementation of configuration storage with better reliability was introduced.
- An issue that users are unable to access the right-click menu of selected favorite tab has been fixed.
- The memory management was optimized.

## Module modifications

### Auto FATE [Script]

- Now the character would only engage special FATEs when the progress is above 20%.

### Auto Display Ready Check Remaining Time [UI_Optimization]

- An issue that the time may not be accurate in some cases has been fixed.

### Auto Redirect Action Target [Action]

- Now the character would also target the entity after redirection.

### Auto Open Treasure Coffer [Combat]

- An issue that the character may be unable to open a coffer in some cases has been fixed.

### Better Enmity List [UI_Optimization]

- The appearance was optimized.

### Better Target Bar [UI_Optimization]

- An issue that some name texts may be incorrect has been fixed.

### Live Anonymous Mode [General]

- The performance was improved.
- The option `Time Obfuscation` was removed since it's now technically unviable.
- The option `Damage Obfuscation` is disabled by default.

### Fast Object Interaction [UI_Optimization]

- An issue that the option `Hide When In Combat` may not work as intended has been fixed.
- An issue that some non-interactive objects due to exceeding the distance were considered as interactive objects has been fixed.
- Now the title bar won't be displayed for the floating window.

### Auto Chat Bot Reply [General]

- The information within `Game Context` - `Area` is more detailed now with comprehensive explanations of area types.
- Now `Status` was added to `Game Context` in order to provide detailed behaviour information like crafting and gathering.
- The replies belong to different sessions would no longer interfere with each other.
- The World Book search was refactored. The logic was adjusted from simple inclusion checks to top-k weighted Tokens using Tokenize.
- An issue that `Enable Filter Model` didn't work has been fixed.
- The overall performance was optimized.

### Fast Teleportation Panel [UI_Optimization]

- The drawing logic of buttons in the side bar was optimized. The consistency of their appearance was improved.
- Now a `/pdrtp` command without a parameter would show or hide the floating window.

### Auto Remove Action Animation Lock [Action]

- Now the duration of the animation lock could be customized.

### Auto Count Players Nearby [General]

- Now a timer will initiate while the character is targeted by others.
- New feature `History Of Being Targeted`, which contains 100 entries that the character was targeted by others, including player names, player servers, areas, time and duration.
- New option `Ignore Friend`, to ignore while character is targeted by a friend. Disabled by default.
