## Modifications on Daily Routines

- A new reliable and efficient implementation has been introduced for `Skip Logos`.
- The computation of the font scaling feature was adjusted. Now it's more inline with that on Dalamud.
- The efficiency and memory footprint of the font building feature was greatly improved.

## Module modifications

### Auto Expert Delivery [UI_Operation]

- The interaction logic of the floating window was optimized.

### Instant Teleportation [System]

- The overall smoothness was improved.
- Some measures to clear teleportation stuck have been introduced.

### Auto Cosmic Fortune [General]

- Now the module works on `Oizys` and for `Oizys Credit`.

### Field Operations Entry Command [System]

- New parameter `oizys` has been introduced for `Oizys` in `Cosmic Exploration`. (`/pdrfe oizys`)
- The drawing logic of the configuration interface was adjusted.

### Auto Display Network Latency [System]

- An issue that information update would only initiate after acquiring IP has been fixed.
- More IP source information is now available to check.

### Better Market Board [UI_Optimization]

- The interface appearance was adjusted.

### Better World Visit [System]

- An issue that, the interface would become inaccessible after launching the game but staying offline for quite some time, has been fixed.

### Auto Hide Pop-ups [UI_Optimization]

- Now the module works for the new game content in `Cosmic Exploration`.
- The drawing logic of the configuration interface was adjusted.

### Auto Gather In The Diadem [Script]

- An issue that, it may be unable to acquire available gathering nodes nearby due to missing case logic for latin letters, has been fixed.
- The overall performance was improved.

### Occult Crescent Helper [Assistance]

- The instance ID is now displayed in a floating window using vanilla game assets rather than in Server Info Bar.
