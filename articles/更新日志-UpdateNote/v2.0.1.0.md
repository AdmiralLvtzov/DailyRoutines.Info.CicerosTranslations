## 本体修改

- 为 `跳过标志画面` 功能换用了更稳定快速的跳过方法
- 调整了界面字体缩放的计算模式, 以和 Dalamud 更加类似
- 大幅提升了字体构建速度, 降低了构建过程中的内存占用

## 删除模块

### 长咏唱技能优化 [技能]

因除极少数技能外, 现在玩家技能均为短咏唱, 模块作用受限, 如有需要请使用新模块 `自定义技能咏唱/复唱时间`

## 新增模块

### 自动过滤日志消息 [系统]

配置完毕后, 自动阻止对应的游戏内日志消息发送与显示

![AutoFilterLogMessage](/assets/Changelog/2.0.1.0/AutoFilterLogMessage.png)

### 自动屏蔽非指令聊天信息发送 [系统]

仅允许发送以半角斜杠开头的命令消息, 自动阻止发送其他所有一般聊天信息, 避免误触或意外发送全角指令等

![AutoGuardChatMessage](/assets/Changelog/2.0.1.0/AutoGuardChatMessage.png)

### 自定义技能咏唱/复唱时间 [技能]

配置完成后, 自动修改指定技能的咏唱/复唱时间

![CustomActionCastRecastTime](/assets/Changelog/2.0.1.0/CustomActionCastRecastTime.png)

## 模块分类调整

### 自动播报技能直暴击情况

`战斗` → `技能`

## 模块修改

### 自动筹备稀有品 [界面操作]

- 优化了悬浮窗交互逻辑

### 即刻传送 [系统]

- 提升了整体传送流畅度
- 针对意外无法传送的情况进行了处理, 避免人物行动被锁死

### 自动宇宙好运道 [一般]

- 支持 `俄匊斯行星` 与 `俄匊斯信用点`

### 特殊场景探索进入指令 [系统]

- 新增 `宇宙探索` 玩法的 `俄匊斯行星` 进入参数 `oizys` (`/pdrfe oizys`)
- 调整了配置界面的绘制逻辑

### 自动显示网络延迟 [系统]

- 修复了 IP 地址信息获取会阻碍延迟信息更新的问题
- 为 IP 地址新增显示更加详细的来源外显

### 更好的市场布告板 [界面优化]

- 调整了界面显示效果

### 快捷跨界传送 [系统]

- 修复了在未登录状态下保持较长时间后再次登录游戏会导致模块界面无法打开的问题

### 自动隐藏横幅 [界面优化]

- 新增了 `宇宙探索` 玩法内的数个横幅
- 调整了配置界面的绘制逻辑

### 自动云冠群岛采集 [脚本]

- 修复了获取附近可用采集点时未对采集点名称字符串比较忽略大小写, 致使在存在大小写的语言中无法获取周围可用采集点的问题
- 优化了整体性能表现

### 蜃景幻界新月岛助手 [辅助]

- 岛 ID 现在显示在新月岛信息界面内, 移除了服务器信息栏中的显示

![OccultCrescentHelper-IslandIDAddon](/assets/Changelog/2.0.1.0/OccultCrescentHelper-IslandIDAddon.png)

- 移除了新月岛信息界面内的长效增益效果技能使用按钮, 改为使用悬浮窗显示在知见水晶上, 对应新配置项 `快捷使用知见水晶`, 默认启用
- 修复了在部分意外区域可以使用命令 `/pdr pbuff` 的问题

### 自动无人岛工房生产计划 [界面操作]

- 支持繁体中文服物品名称格式
- 修改模块权限为 `特供 (国服)`、`特供 (繁中服)`
- 优化重构了模块逻辑

### 快速目标交互 [界面优化]

- 飞行状态下将自动忽略物体与本地玩家间垂直高度, 以避免欲交互物体被过滤
- 修改了物体对象与本地玩家间距离计算逻辑
- 优化了飞行状态下交互的处理逻辑

### 敌对列表界面优化 [界面优化]

- 修复了在 PVP 副本内仍会显示相关状态效果的问题
- 调整了部分节点大小位置的计算逻辑
