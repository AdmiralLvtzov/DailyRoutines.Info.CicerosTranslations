## Modifications on Daily Routines

- The storage logic and memory footprint of preset data have been optimized.
- A thread security issue while updateing module data in the main interface has been fixed.
- The memory performance of some dependencies has been optimized.
- Some jarring indention of text elements in vanilla-style interfaces has been fixed.
- Now a recursive prerequisite check would be performed while enabling a prerequisite module.
- An issue within the interception method of some data packets has been fixed.修复了错误的游戏数据包拦截处理方法

## Module modifications

### Better Crafting Log [UI_Optimization]

- An issue that currency icons in `All Vendors` were fragmentary has been fixed.
- A null exception after closing the `Crafting Log` interface has been fixed.
- Now `Auto Display Available NPC Vendor` is a a prerequisite module.
- Now loading of vanilla-style UI elemnts would be hold until the interface of `Crafting Log` is fully available.

### Player Target Info Expand [UI_Optimization]

- Omitted localization texts have been supplemented.
- The logic of interface drawing has been refactored.

### Auto Countdown Notification [Notification]

- An issue that the module didn't work has been fixed.

### Auto Ready Check Notification [Notification]

- The module has been refactored adopting a more efficient and reliable approach.

### Auto Recruitment End Notification [Notification]

- The module has been refactored adopting a more efficient and reliable approach.
- Now no notification would be sent if users end the party recruitment manually.

### Frontline Map Radar [Combat]

- The logic of updating player information has been optimized.
- Now the offset and scale of marking texts could no longer be customized.

### Better Search In Right-click Menu [UI_Optimization]

- Now reborn `Warrior of Light` (A simplified Chinese online website for sharing glamour collections among players) is supported.

### Auto Transmute Materia [UI_Operation]

- An issue that some UI elements may be missing when there's no UI mod installed has been fixed.

### Auto Sightseeing Log [Script]

- New configuration option `Always Wayfinding`. Enabled by default.
- The logic of character navigation has been fully refactored.
- The coordinate of `Gold Court` has been amended.
- The logic of determining weather in a specific zone has been amended.
